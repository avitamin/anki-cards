# Anki Cards — Контекст проекта

## Обзор директории

Эта директория содержит коллекцию **Anki-карточек** для изучения языка программирования **Go (Golang)**. Карточки созданы на основе профессиональных книг по Go и организованы по уровням сложности.

## Структура

```
anki-cards/
├── README.md                          # Пустой заголовок проекта
├── go/
│   └── anki_go_cards.tsv              # TSV-файл с карточками (26 карточек)
├── LLM/
│   └── chatgpt-project-sources-2-cards.md  # Промпт/инструкция для генерации карточек
└── QWEN.md                            # Этот файл
```

## Ключевые файлы

### `go/anki_go_cards.tsv`

TSV-файл с Anki-карточками по Go. Содержит **26 карточек** с полями:

| Поле | Описание |
|------|----------|
| **Front** | Вопрос на русском языке |
| **Back** | Ответ с HTML-разметкой (`<br>`, `<b>`) |
| **Tags** | Теги для классификации |

**Тематика карточек:**
- Инструменты Go (`go fmt`, `go vet`, `go run`, `go build`, `staticcheck`)
- Дизайн API (принимайте интерфейсы, возвращайте структуры)
- Обработка ошибок (ошибки как значения, обёртка ошибок)
- Конкурентность (`errgroup`, `WaitGroup`, `context`)
- Конфигурация (паттерн функциональных опций, cobra + viper)
- CLI-разработка (cobra, флаги)

**Уровни сложности** (теги `level_*`):
- `level_1_foundations` — основы
- `level_2_core_understanding` — ядро
- `level_3_practical_usage` — практическое применение
- `level_4_tradeoffs` — компромиссы
- `level_5_system_thinking` — системное мышление

**Источники** (книги):
- «Go идиомы и паттерны проектирования» — Джон Боднер, 2-е изд.
- «Golang для профи» — Цукалос, 3-е изд.
- «100 ошибок Go и как их избежать» — Харшани

### `LLM/chatgpt-project-sources-2-cards.md`

Детальный промпт для мультиагентной системы генерации Anki-карточек. Описывает:

- **RAG pipeline**: discovery → structure extraction → knowledge extraction → global registry → coverage check → curriculum → card generation → export
- **Жёсткие ограничения**: карточки только на основе цитат из файлов, без внешних знаний
- **Роли агентов**: Mentor, Curriculum Designer, Cognitive Scientist, Domain Expert, Reviewer
- **Формат экспорта**: TXT с `;`-разделителем, без HTML, с тегами
- **Ментальные модели**: Pareto, First Principles, Chunking, Retrieval Practice, Desirable Difficulty и др.

## Использование

### Импорт карточек в Anki

1. Откройте Anki → `File` → `Import`
2. Выберите `go/anki_go_cards.tsv`
3. Убедитесь, что разделитель — `Tab`, поля: Front, Back, Tags
4. Импортируйте

### Генерация новых карточек

Файл `LLM/chatgpt-project-sources-2-cards.md` содержит готовый промпт для генерации карточек из новых книг. Для этого:

1. Поместите PDF/текст книги в директорию проекта
2. Запустите LLM с промптом из `chatgpt-project-sources-2-cards.md`
3. Система извлечёт знания, построит реестр и создаст карточки по уровням

## Технологии и инструменты

- **Anki** — система интервального повторения
- **TSV формат** — табличный формат для импорта (Tab-Separated Values)
- **Go (Golang)** — язык программирования, предмет изучения

## Конвенции

- Карточки на **русском языке**
- Ответы содержат HTML-разметку для форматирования
- Каждая карточка привязана к источнику (файл, глава, подглава, цитата)
- Теги нормализованы: `go`, `backend`, `anki_cards`, `level_*`, тематические теги
