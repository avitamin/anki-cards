# anki-cards

Коллекция Anki-карточек для изучения Go на основе профессиональной литературы.

## Что в репозитории

- `go/anki_go_cards.tsv` — экспорт колоды Anki в TSV-формате.
- `LLM/chatgpt-project-sources-2-cards.md` — промпт и workflow для генерации карточек из исходных материалов.
- `QWEN.md` — краткий контекст проекта и описание структуры.
- `AGENTS.md` — инструкции для контрибьюторов.

Сейчас в колоде Go — 26 карточек с полями `Front`, `Back`, `Tags`.

## Как импортировать в Anki

1. Откройте Anki.
2. Выберите `File` -> `Import`.
3. Укажите файл `go/anki_go_cards.tsv`.
4. Проверьте настройки импорта:
   - разделитель: `Tab`
   - поля: `Front`, `Back`, `Tags`
5. Подтвердите импорт.

## Как устроены карточки

- Вопросы и ответы написаны на русском.
- Ответы могут содержать HTML-разметку (`<br>`, `<b>`).
- Каждая карточка содержит source traceability: файл, главу, подглаву и цитату.
- Для классификации используются теги, например: `go`, `anki_cards`, `level_1_foundations`.

## Как генерировать новые карточки

1. Подготовьте исходные материалы в проекте.
2. Используйте промпт из `LLM/chatgpt-project-sources-2-cards.md`.
3. Проверьте, что новые карточки:
   - основаны на цитатах из файлов проекта
   - покрывают несколько глав и тем
   - экспортированы в TSV с корректными тегами
